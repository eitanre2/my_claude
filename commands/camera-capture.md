Capture a clip from the connected camera and send it to the session.

Use the camera-capture skill. Default to video mode, 5 seconds, device 0 (HD Pro Webcam C920).
If arguments are provided (e.g. `/camera-capture photo` or `/camera-capture 10`), honour them:
- a number = duration in seconds
- "photo" = still image instead of video
- a number like "1" after "video" or as the only arg = duration
